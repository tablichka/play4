*Искусство рыбалки:* Выбор наживки Существует три типа наживки. Гильдия рыбаков делит наживки на зеленые, коричневые, желтые, по глубине погружения в воду и по уровню. Разные типы рыб имеют разные привычки. Вы можете ловить разые типы рыб используя разные типы приманки. Второй уровень мастерства рыбалки учит правильно выбирать приманку.
