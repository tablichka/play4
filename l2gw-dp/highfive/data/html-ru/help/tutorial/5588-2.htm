[player_help help/5588.htm|Список] *Камера* Движение мыши с зажатой правой кнопкой меняет положение камеры. Это полезно для осмотра местности при охоте. Нажатие правой кнопки возвращает стандартное положение камеры. Используйте кнопки Up/Page Down для приближения и отдаления камеры. [player_help help/5588.htm|Список]
