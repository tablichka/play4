Семя Жизни:
Эта наполненная огненными цветками необычная сфера, воздействуя на внутренние стены Семени, создала исскуственное Семя Жизни, идентичное Проклятому Семени Жизни.
Сейчас огненные цветки пляшут внутри сферы, издающей низкие искаженные звуки.
Посылая безвольной нежити фальшивые сигналы, она будет их приманивать.
